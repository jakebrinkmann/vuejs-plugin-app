# vuejs-plugin-app

This application aims to import components from [vuejs-plugin-01][0], 
as a learning exercise on making extensible modules in VueJS. 

## Vue devtools:

In the console, with [Vue devtools][2]
```javascript
$vm0.newItem = 'Hi'
```
